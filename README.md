# Original Image Extractor
link for the figma plugin 
https://www.figma.com/community/plugin/1557149400939799570

Figma plugin to extract original, unedited source images at full resolution with preserved quality.

## Features

- Scans current page for all images
- Extracts images in their original format (PNG, JPG, WebP, GIF, AVIF, etc.)
- Converts images to multiple formats (PNG, JPG, WebP, GIF, AVIF)
- Preserves original quality and resolution
- Shows image previews and metadata
- Bulk download with progress tracking

## Installation

1. Download the plugin files
2. In Figma, go to Menu → Plugins → Development → Import plugin from manifest
3. Select the manifest.json file

## Usage

1. Open a Figma file with images
2. Run the plugin from Menu → Plugins → Original Image Extractor
3. Click "Scan Current Page" to find all images
4. Select images you want to download
5. Choose export formats (if converting)
6. Click "Download Originals" or "Convert & Download"

## Supported Formats

- PNG, JPG/JPEG, WebP, GIF, AVIF, BMP, TIFF, SVG
